# 老人问题

一个面向微信公众号写作的 Codex Skill，用来创作、改写、规划或点评银发/养老议题内容。

它适合处理这些方向：

- 养老焦虑、退休生活、独居老人
- 子女边界、带孙边界、家庭照护
- 存款、房子、养老钱、房产过户
- 养儿防老、养老院、老伴走后等反共识选题
- 朋友式、故事驱动、情绪克制的公众号文章

## 安装

把本仓库复制到 Codex skills 目录：

```text
C:\Users\<你的用户名>\.codex\skills\老人问题
```

重启 Codex 后即可使用。

## 使用

```text
Use $老人问题 写一篇公众号文章：

主题：70岁以后才明白，养儿防老不如先给自己留一手
```

也可以用于选题规划：

```text
Use $老人问题 给我 20 个关于老人存款边界、房产过户和子女养老责任的公众号选题。
```

## 输出风格

- 具体年龄、具体金额、具体家庭场景
- 朋友式提醒，不用专家口吻
- 反共识但不煽动对立
- 短段落，适合手机阅读
- 以真实故事和行动提醒收束
